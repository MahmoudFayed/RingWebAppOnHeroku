The Ring Programming Language

Binary release for Ubuntu Linux 64 bit
===================================================

To install 




(1) Install Libraries
	cd ring/src
	./installdep.sh




(2) To be able to call ring from any folder 
	
	cd ring
/bin
	sudo ./install.sh
	


(3) Run Ring Notepad
	
	
	cd ring/applications/rnote
	
	ring rnote.ring



Start learning from the documentation
